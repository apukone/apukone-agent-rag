
# RAG Demonstration

This is a sample document to test the Retrieval Augmented Generation (RAG) capabilities of the Apukone RAG Agent.

## What is Apukone?
Apukone is a multi-agent system designed to handle complex tasks using specialized agents.

## Agents
Currently, there are several known agents:
1. **NHL Agent**: Uses real-time NHL data to answer hockey questions.
2. **Weather Agent**: Provides weather forecasts.
3. **RAG Agent**: Uses this document to answer questions!

## Key Features
- **MCP Support**: Agents use the Model Context Protocol to define tools.
- **Local Inference**: Uses Ollama for LLM and Embeddings.
- **Vector Search**: Uses LanceDB for storing knowledge.
